# Cyber Hacking Competition Preparation Course

Welcome to the Cyber Hacking Competition Preparation Course. This repository contains lecture materials, practice exercises, and resources designed to prepare students for Capture The Flag (CTF) competitions.

## Course Structure

- **Lectures:** Comprehensive guides on various cybersecurity topics.
- **Practices:** Hands-on exercises to reinforce learning.
- **Resources:** Lists of essential tools, skills, and tips for competition success.

## Getting Started

Start by reading through the lectures in order, and then move on to the corresponding practice exercises.
