# Review: Skills, Tools, & Knowledge

## Essential Toolkit
A checklist of tools you should have installed and be familiar with.

### General
- [ ] **Linux Terminal:** bash/zsh proficiency.
- [ ] **Python:** For scripting solutions.
- [ ] **Git:** Version control.
- [ ] **Virtual Machine:** VirtualBox or VMware with Kali Linux.

### Web
- [ ] **Burp Suite:** Web proxy and scanner.
- [ ] **Firefox:** With FoxyProxy and Wappalyzer extensions.
- [ ] **Gobuster/Dirb:** Directory brute-forcing.
- [ ] **SQLMap:** Automated SQL injection.

### Forensics & Stego
- [ ] **Exiftool:** Metadata extraction.
- [ ] **Binwalk:** Firmware analysis and extraction.
- [ ] **Steghide / Stegsolve:** Steganography tools.
- [ ] **Wireshark:** Network packet analysis.
- [ ] **Strings:** Extract text from binary.

### Binary & Reverse Engineering
- [ ] **GDB (with GEF or Pwndbg):** Debugging.
- [ ] **Ghidra:** Decompiler.
- [ ] **Radare2:** Command-line framework.
- [ ] **Pwntools:** Python library for CTF exploit writing.

### Cryptography
- [ ] **CyberChef:** The "Swiss Army Knife" of crypto.
- [ ] **John the Ripper / Hashcat:** Password cracking.

## Knowledge Check
Can you explain these concepts?
- [ ] CIA Triad (Confidentiality, Integrity, Availability)
- [ ] How a Buffer Overflow works
- [ ] The difference between Encoding, Hashing, and Encryption
- [ ] How the HTTP Request/Response cycle works
- [ ] What a "Shebang" (`#!/bin/bash`) is
